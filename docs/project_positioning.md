# EdgeTalk 项目定位

## 项目名称

EdgeTalk：面向工业设备维护场景的边缘端企业知识库 AI 助手

## 项目定位

EdgeTalk 是一个面向制造业自动化产线设备维护场景的 AI 助手原型，计划部署在 Jetson Orin Nano 边缘设备上。

本项目不是泛用聊天机器人，而是一个面向 ToB 企业知识库和工业现场维护场景的 AI 解决方案 Demo。

## 目标用户

- 设备维护人员
- 产线操作员
- 现场工程师
- 新入职维修人员
- 工厂设备管理人员

## 业务痛点

1. 设备资料分散，查找效率低。
2. 故障处理依赖老师傅经验，新员工上手慢。
3. 现场人员不方便长时间打字查资料。
4. 工厂网络环境可能不稳定。
5. 企业内部设备资料不适合全部上云。
6. 故障处理过程缺少统一记录，不利于知识沉淀。

## 解决方案

EdgeTalk 通过本地知识库、RAG 检索、Agent 工具调用、SQLite Memory 和 FastAPI 接口，为工业现场提供本地化设备维护问答能力。

## 核心功能

1. 设备手册问答
2. 故障代码查询
3. 维修 SOP 查询
4. 巡检规范查询
5. 安全规范查询
6. 多轮故障排查
7. Jetson 边缘部署验证

## 业务价值

1. 提升设备资料查询效率。
2. 降低新员工培训成本。
3. 减少对个人经验的依赖。
4. 支持本地化部署，降低资料上云风险。
5. 为制造业客户提供 AI 知识库 PoC Demo。

## 当前限制

1. 当前知识库为模拟资料，尚未接入真实企业文档。
2. 当前 Agent 采用轻量规则路由。
3. Jetson 部署和性能测试将在后续完成。
4. ASR/TTS 在 Jetson Linux 环境下仍需适配。
